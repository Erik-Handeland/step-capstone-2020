IP Addresses
An Internet Protocol address (or IP address) is the name associated with a device that's connected to a network. Think of IP addresses as a sort of network ID; packets are sent and received from IP addresses in the same way mail is sent and received from different home addresses. IPv4 addresses, which consist of four bytes of numbers, are sorted into one of five classes: Class A, Class B, Class C, Class D, and Class E.

Class A
1.0.0.0 to 126.255.255.255
Class B
128.0.0.0 to 191.255.255.255
Class C
192.0.0.0 to 223.255.255.255
Class D
224.0.0.0 to 239.255.255.255
Class E
240.0.0.0 to 254.255.255.255


Different classes are assigned to different types of users. Class A typically serves large companies and organizations, while Class B serves smaller ones. Class C is typically used by private, individual users. Class D is reserved for mutlicasting while Class E is reserved for special future purposes.Take a look at your file data and see how the addresses you connect to are distributed across the classes and addresses.

TCP Vs. UDP
TCP and UDP are two fundamental protocols used to transmit data. TCP packets retransmit themselves if a packet gets lost. UDP packets don't retransmit themselves so they travel faster then TCP.
Packet Traffic
